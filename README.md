# Metallurgical CFD AI Workflow

This repository is an early-stage research workflow for AI-assisted metallurgical CFD/CPFD post-processing, simulation data analysis, and report drafting.

## Background

I am a graduate student in metallurgical engineering. My current research interest focuses on digitalization and intelligent modeling of metallurgical processes, including CFD/CPFD simulation, multiphase flow analysis, Python-based data processing, and machine learning-assisted modeling.

## Project Goal

The goal of this project is to build a lightweight Python workflow that can assist with:

- Reading exported CFD/CPFD simulation data
- Cleaning and organizing flow-field data
- Plotting key simulation indicators
- Generating preliminary simulation analysis reports
- Supporting literature-based model parameter extraction
- Exploring AI-assisted coding workflows using large language model APIs

## Current Features

At this stage, the repository contains simple prototype scripts and example files, including:

- A sample CFD data table
- A Python script for reading simulation data
- A Python script for plotting key variables
- A draft outline for simulation report generation

## Planned Work

Future work will include:

- Supporting Fluent and Tecplot exported data formats
- Adding batch post-processing functions
- Building reusable plotting templates for CFD reports
- Testing LLM-assisted coding and report generation workflows
- Exploring surrogate modeling for metallurgical process simulation data

## Application Scenario

This repository is intended for research and engineering learning scenarios related to metallurgical process simulation, especially for combining CFD/CPFD numerical simulation with AI-assisted data analysis and code generation.
