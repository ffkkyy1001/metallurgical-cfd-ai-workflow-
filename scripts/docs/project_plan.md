# Project Plan

## Short-term Goal

Build a simple Python workflow for CFD/CPFD data reading, cleaning, plotting, and report drafting.

## Medium-term Goal

Extend the workflow to support common exported data from Fluent, Tecplot, or CPFD simulation tools.

## Long-term Goal

Explore the integration of large language model APIs for simulation data interpretation, code generation, and AI-assisted report writing.

## Expected Output

- Python scripts for CFD post-processing
- Example simulation data processing workflow
- Visualization templates
- Draft simulation report structure
- Notes on AI-assisted research workflow
